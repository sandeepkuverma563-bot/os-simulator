# OS Simulator

A web-based Operating System simulator built to understand and visualize core OS concepts in a practical way.

It helps in learning CPU scheduling, memory management, and deadlock concepts through interactive execution and comparison of algorithms.

---

## Features

- CPU Scheduling algorithms (FCFS, SJF, SRTF, Priority, Round Robin)
- Memory Management (FIFO, LRU, Optimal Page Replacement)
- Memory Allocation techniques (First Fit, Best Fit, Worst Fit)
- Banker's Algorithm for deadlock avoidance
- Performance comparison between algorithms
- Interactive simulation and step-by-step visualization

---

## Tech Stack

Frontend: React, Vite, Tailwind CSS  
Backend: Node.js, Express  
API Communication: Axios  
Charts: Recharts  

---

## Folder Structure

Frontend has all UI components, pages, and styles.  
Backend contains API routes, controllers, and OS algorithm logic.  

---
## How to Run Locally

Install dependencies:

npm install  
npm run install:all  

Start project:

npm run dev  

Frontend:
http://localhost:3000  

Backend:
http://localhost:5000
